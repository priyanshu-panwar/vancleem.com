Revenue Management Solution

Developed a Revenue Management Analytics Solution using Microsoft Access + VBA. The Microsoft Access application forecasts fleet utilization over 60 days and compares the data against historical utilization. The solution helped significantly reduce utilization gaps along with increasing revenue and margins.

Yield Management Solution

Developed an automated yield management solution using Microsft Excel + VBA. The Yield Manager updates date ranges from a calendar by applying the corresponding color-coded rate levels. The new rates are automatically distributed in various formats (xls & pdf) across multiple channels via email, Dropbox and FTP.

Online Web Directory

Launched an online web directory for businesses in the French Overseas Departments. The back-end allows the administrator to view website performance statistics and contains a CRM to manage communications with registered businesses and prospects. Email automation is used based on the various stages of the sales cycle.
